# エクスポートファイル格納ディレクトリ

このディレクトリには、ローカルアプリからエクスポートしたカードデータを配置します。

## 使い方

1. ローカルの完全版アプリでデータをエクスポート
2. ダウンロードされたJSONファイル（例: `chizai-cards-2026-02-15-173045.json`）を**そのまま**このディレクトリに配置
3. GitHubにコミット＆プッシュ
4. GitHub Actionsが自動的に最新ファイルを`public/data.json`にコピーしてデプロイ

## 注意事項

- ファイル名は `chizai-cards-YYYY-MM-DD-HHmmss.json` の形式
  - 例: `chizai-cards-2026-02-15-173045.json`（2026年2月15日 17:30:45にエクスポート）
- GitHub Actionsはファイル名の逆順（最新の日時）で自動的に選択します
- 同じ日に複数回エクスポートしても、時刻で区別されます
- 過去のエクスポートファイルも履歴として残せます
