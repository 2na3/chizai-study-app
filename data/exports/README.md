# エクスポートファイル格納ディレクトリ

このディレクトリには、ローカルアプリからエクスポートしたカードデータを配置します。

## 使い方

1. ローカルの完全版アプリでデータをエクスポート
2. ダウンロードされたJSONファイル（例: `chizai-cards-2026-02-15.json`）を**そのまま**このディレクトリに配置
3. GitHubにコミット＆プッシュ
4. GitHub Actionsが自動的に最新ファイルを`public/data.json`にコピーしてデプロイ

## 注意事項

- ファイル名は `chizai-cards-YYYY-MM-DD.json` の形式を推奨
- GitHub Actionsは日付が最新のファイルを自動的に選択します
- 過去のエクスポートファイルも履歴として残せます
